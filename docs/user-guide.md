# 用户使用指南

这份指南说明如何在桌面灵里创建、配置和使用自己的桌宠。桌面灵默认不自带角色模型、声音模型和 API Key；你需要导入自己拥有使用权的 Live2D 模型，并按需要配置聊天和语音能力。

## 1. 打开桌面灵

安装后运行“桌面灵”。打开后会进入主页。

主页左侧是本机已经创建的桌宠列表，右侧是当前选中桌宠的详情区。

你可以在主页完成这些操作：

- 查看已有桌宠
- 创建新桌宠
- 查看桌宠详情
- 预览已导入的 Live2D 模型
- 启用或关闭桌宠窗口
- 进入编辑器修改配置
- 连接或关闭声音模型
- 删除本地桌宠

如果某个桌宠显示“待导入”，说明它还没有可用的 Live2D 模型。你需要先进入编辑器导入模型，导入成功后才能启用。

## 2. 创建新桌宠

在主页点击“创建新桌宠”，会进入“桌宠设置”页面。

设置页面左侧包含这些栏目：

- 基础信息
- Live2D
- 界面主题
- AI设置
- 语音系统

其中“AI设置”下包含：

- LLM配置
- 角色人设
- 事件配置
- 表现映射

“语音系统”下包含：

- 语音输入
- 声音模型

新桌宠必须先保存“基础信息”。保存成功后，软件才会为它创建本地数据目录，之后才能继续导入 Live2D、配置 AI、保存表现映射、保存事件和配置语音。

如果当前页面有未保存修改，切换栏目或返回主页时会出现提醒。确认离开会丢失当前页面尚未保存的内容。

## 3. 基础信息

基础信息会显示在主页列表和详情区。

必填内容：

- 名称

可选内容：

- 头像
- 简介
- 角色定位
- 性格描述
- 适合场景

头像支持：

- PNG
- JPG
- JPEG
- WebP

选择头像后会进入裁剪窗口。拖动蓝色方框可以调整裁剪位置，滚动鼠标滚轮可以调整裁剪大小。确认后点击“保存头像”。

“适合场景”可以选择已有标签，也可以输入自定义标签后点击“添加场景”。

填写完成后点击“保存”。保存成功后，这个桌宠会出现在主页列表里。

## 4. Live2D 导入

进入“Live2D”页面后，可以导入模型。

你需要选择的是完整 Live2D 模型文件夹，不是单独选择入口文件。可以点击“选择文件夹”，也可以把模型文件夹拖入导入区域。

当前支持：

- Cubism 2：`model.json` / `.moc`
- Cubism 3 / 4 / 5：`.model3.json` / `.moc3`

导入流程分为三步：

1. 选择模型文件夹
2. 检查模型资源
3. 保存桌宠

选择文件夹后，软件会自动检查：

- 模型入口
- Moc 文件
- 贴图
- 动作
- 表情

资源检查通过后，可以点击“预览模型”。预览中可以确认：

- 模型是否正常显示
- 贴图是否完整
- 模型大小和位置是否合适
- 动作是否能播放
- 表情是否能切换

预览区里可以点击动作或表情进行测试，也可以点击“重置状态”让模型回到默认状态。确认没有问题后，点击“保存”。

保存成功后，Live2D 页面会显示“Live2D 模型已导入”，并显示模型入口、贴图数量、动作数量和表情数量。已导入后仍然可以：

- 导入新模型
- 测试动作/表情
- 返回已导入模型

如果模型文件夹缺少入口声明，页面可能会出现“自动声明”按钮。点击后，软件会尝试为当前文件夹生成可用入口。

## 5. 界面主题

“界面主题”用于设置这只桌宠的界面外观。

主题会影响：

- 右键快捷菜单
- 聊天框
- 字幕气泡

内置主题包括：

- 软糖风
- 摇滚风
- 像素风
- 手账风
- 赛博风
- 极简风

选择主题后点击“保存”。主题是按桌宠单独保存的，不同桌宠可以使用不同主题。

如果你有主题 JSON 文件，也可以点击“导入主题”。导入后，新主题会出现在主题列表最前面，选中后再点击“保存”即可使用。

## 6. LLM 配置

进入“AI设置 / LLM配置”，可以配置文字聊天模型。

你需要填写：

- 服务商
- Base URL
- API Key
- 模型名

服务商可以选择预设项，也可以选择“自定义”后手动填写兼容 OpenAI 接口的地址。

点击“连接”后，软件会尝试读取可用模型列表。如果读取成功，可以从下拉列表选择模型；如果没有读取到列表，也可以手动填写模型名。

点击“保存设置”后，这只桌宠会保存当前 AI 配置。API Key 只保存在本机，重新填写可以覆盖旧 Key。

如果连接或保存失败，优先检查：

- Base URL 是否正确
- API Key 是否有效
- 模型名是否存在
- 当前网络或本地代理是否可用

## 7. 角色人设

进入“AI设置 / 角色人设”，可以填写桌宠的人设。

主要输入框是“我是谁”。这里可以写：

- 桌宠身份
- 性格
- 说话方式
- 和用户的关系
- 喜欢或讨厌的事物
- 回答时需要遵守的边界

你还需要选择“文字输出语言”：

- 中文
- 日语
- 英语

“你希望回复长度”可以选择：

- 短
- 中
- 长

回复长度可以不勾选。保存后，这些内容会影响桌宠聊天时的回答风格。

## 8. 表现映射

进入“AI设置 / 表现映射”，可以把 Live2D 模型里的动作和表情整理成桌宠能理解的表现。

使用前请先完成 Live2D 导入。

导入模型时，软件会自动读取其中的：

- 动作源文件
- 表情源文件

识别完成后，页面会分成“动作”和“表情”两个列表。每一行包含：

- 源文件名
- 桌面预览按钮
- 映射 key
- 动作描述或表情描述

源文件名由导入时自动识别，不能手动修改。点击“预览”会打开或切换到当前桌宠的桌面窗口并播放该资源；你可以边看效果边填写映射 key 和中文描述。

示例：

```text
angry01.mtn          [预览]    key: angry      描述：生气皱眉
smile.exp3.json      [预览]    key: happy      描述：开心微笑
idle.motion3.json    [预览]    key: idle       描述：待机动作
```

映射 key 是给事件配置和 AI 回复引用的名称。建议使用简短英文，例如：

- normal
- happy
- angry
- surprised
- shy
- nervous
- panic
- thinking
- idle

映射 key 必须以英文字母开头，只能包含英文、数字、下划线或短横线，并且不能重复。

中文描述用来告诉你和 AI 这个表现是什么意思。建议写具体一点，例如“开心微笑”“慌张后退”“认真思考”“生气皱眉”。

不想使用的源文件可以留空。只有填写了有效 key 和描述的行，才会作为可用表现保存。

表现映射页面还有“随机”功能。开启后，AI 回复时会在你选择的范围内随机播放动作或表情。随机范围可以选择：

- 动作
- 表情
- 全部

开启随机后，下方映射编辑会暂时锁定。关闭随机后，AI 回复会重新按映射描述选择表现。

## 9. 事件配置

进入“AI设置 / 事件配置”，可以为桌宠的不同行为设置反馈。

使用前请先完成 Live2D 导入。动作和表情源会在导入时自动识别；在事件配置中选择资源后，可点击“预览”直接在桌面桌宠中确认效果。

当前可以配置的事件包括：

- 模型加载完成
- 点击模型
- 连续快速点击
- 拖拽模型
- 打开聊天
- 关闭聊天
- 开启穿透
- 关闭穿透
- 长时间无操作
- 关闭桌宠

每个事件可以设置：

- 触发动作
- 触发表情
- 表情持续时间
- 候选台词

“动作”和“表情”来自已经保存的表现映射。候选台词可以写多条，事件触发时软件会从中随机选择一条显示。

如果某个事件不需要动作、表情或台词，可以留空。

## 10. 启用桌宠

回到主页，点击一个已经导入 Live2D 模型的桌宠。

详情区会显示：

- Live2D 预览
- 当前启用状态
- 声音模型状态
- 界面主题
- 桌宠详情

点击“启用桌宠”后，会打开透明无边框桌宠窗口。

桌宠窗口中右键点击模型区域，可以打开圆形快捷菜单。常用操作包括：

- 对话：打开或关闭聊天框
- 触控：开启后可以拖动桌宠窗口
- 穿透：让桌宠尽量不阻挡桌面操作
- 关闭：关闭桌宠窗口
- 中心按钮：收起菜单

开启“穿透”后，其他菜单按钮会隐藏，只保留“解锁”。再次点击“解锁”后，桌宠会恢复正常交互。

## 11. 聊天

启用桌宠后，右键打开快捷菜单，点击“对话”即可打开聊天框。

如果已经配置好 LLM，输入文字后桌宠会根据人设回复。

如果还没有配置 AI，聊天可能无法得到回复。请先到“AI设置 / LLM配置”保存可用的模型连接。

如果已经配置表现映射，AI 回复时可以根据语境触发表情或动作；如果开启随机表现，则会按随机范围播放动作或表情。

## 12. 语音输入

进入“语音系统 / 语音输入”，可以配置语音转文字。

语音输入是可选功能。未配置时，文字聊天仍可正常使用。

当前语音输入使用腾讯云实时语音识别。你需要填写：

- AppID
- SecretId
- SecretKey

填写后点击“连接”。连接成功后，可以继续配置语音输入参数。

可设置项包括：

- 自动识别说话结束
- 静音超过 1 / 2 / 3 秒自动结束
- 音量阈值
- 回复后继续录音

普通房间可以先使用 18% 左右的音量阈值。环境越吵，阈值可以调高；说话声音越轻，阈值可以调低。

保存后，聊天框中会显示语音输入入口。语音输入需要你主动触发，软件不会后台偷偷录音。

## 13. 声音模型

进入“语音系统 / 声音模型”，可以配置语音回复。

语音回复是可选功能。未配置时，桌宠仍可正常文字聊天和显示字幕。

语音回复需要你准备本机 GPT-SoVITS 环境和声音资源。

你需要填写或选择：

- GPT-SoVITS 本地路径
- SoVITS 模型
- GPT 模型
- 参考音频
- 参考文本

选择声音资源后点击“连接”。连接成功后，页面会自动打开“启用语音回复”。

可设置项包括：

- 输出语言：中文、日语、英语
- 播放模式：逐句播放、完整生成后播放
- 语音与文字一起输出

如果开启“语音与文字一起输出”，软件会等待语音生成完成后，再同时显示文字并播放语音。

如果声音模型已经配置完整，也可以在主页详情区点击“声音模型”直接连接或关闭声音。

连接失败时，优先检查：

- GPT-SoVITS 本地路径是否正确
- SoVITS 模型是否存在
- GPT 模型是否存在
- 参考音频是否存在
- 参考文本是否填写
- 本地 GPT-SoVITS 服务是否可用

语音失败不会影响文字聊天和字幕显示。

## 14. 删除桌宠

在主页选中桌宠后，可以点击详情区里的“删除”。

删除会移除这只桌宠的本地配置和相关本地资源。删除前请确认已经备份需要保留的模型、头像、台词、人设或语音配置。

## 15. 本地数据与隐私

桌面灵是本地优先的软件。

你创建的桌宠、头像、Live2D 模型、AI 设置、人设、表现映射、事件台词和语音设置都会保存在本机。

软件默认不会上传这些内容。更换电脑或重装系统前，请自行备份需要保留的桌宠数据。

请不要把以下内容发布到公开仓库或分享给不可信的人：

- API Key
- 腾讯云 SecretId / SecretKey
- 未授权 Live2D 模型
- 声音模型
- 参考音频
- 本地绝对路径配置
